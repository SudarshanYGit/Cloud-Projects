<h1><b>REAL-LIFE AWS PROJECT</b></h1><br>
<b>AWS EC2 Cost Estimation using Python</b><br>
Built a Python-based AWS EC2 cost estimation tool, This project is a real-life tool built using Python and Boto3. It fetches live EC2 instance details from AWS and calculates the actual running cost and estimated monthly cost in INR. The project demonstrates practical cloud cost monitoring, IAM-based secure access, and AWS cost-optimization best practices.<br>

<b>Key Features</b><br>

Fetches real EC2 instance details using Boto3<br>
Displays instance ID, type, state, and launch time<br>
